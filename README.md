# ApoRaviz_Portfolio

Design with AI. Built as an Angular 21 personal portfolio for Tanonchai Promsiri.

## Stack

- Angular 21
- Node 24
- TypeScript 5.9
- Tailwind CSS v4
- Angular SSR / prerender
- Standalone components and signals

## Development

Use Node 24 before running the project:

```bash
nvm use 24
npm install
npm run start
```

Open `http://localhost:4200/`.

## Build

```bash
npm run build
```

## Test

```bash
npm run test
```
